StudyOS – Final Master Prompt (Production Ready)

Project Overview

Build a production-ready, premium, responsive, full-stack web application named StudyOS.

StudyOS is my personal study operating system. It should help me organize my complete academic life in one place by managing my syllabus, tracking progress, planning studies, recording focus sessions, managing tasks, monitoring statistics, and securely storing all data.

The application should feel like a premium desktop application rather than a simple website.

The UI should be minimal, modern, professional, distraction-free, highly customizable, and extremely fast.

Login System

The website URL should be public.

Anyone can open the website.

However,

No one should be able to enter the application without authentication.

When the website opens:

Display only a Login Screen.

The user must enter a Master Password.

If correct:

Enter StudyOS.

If incorrect:

Display an error.

Keep the user logged in securely until logout.

Provide

* Login
* Logout

Store the password securely using environment variables.

Never hardcode the password.

Final Objective

StudyOS should feel like a professional personal operating system dedicated to studying. It should provide an intuitive, distraction-free workspace where I can manage my syllabus, monitor three separate progress streams (School, Online Class, and Self Study), organize tasks, maintain a calendar of important events, use integrated focus timers, review statistics, earn productivity-based achievements, and securely back up and restore all data. The application must be fully responsive, accessible from any device through a public URL, protected by a secure master password, automatically save all changes, and provide a polished, production-quality experience suitable for daily long-term use.
